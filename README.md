# 🚀 Lógica de Programação com JavaScript

Este repositório contém meus exercícios e desafios práticos desenvolvidos durante a formação **QA Experience** da [DIO.me](https://www.dio.me/).

## 📌 Objetivo
O foco destes estudos é dominar os fundamentos da programação, como variáveis, estruturas condicionais e operadores lógicos, que são a base para a **automação de testes**.

## 💻 Tecnologias Utilizadas
* **JavaScript** (Lógica de programação)
* **Node.js** (Execução do código)
* **VS Code** (Ambiente de desenvolvimento)
* **Git & GitHub** (Controle de versão)

## 📂 Estrutura do Repositório
* `desafios.js`: Primeiros passos com variáveis e saída de dados.
* `desafios2.js`: Manipulação de variáveis `let` e `const` com reatribuição.
* `index.html`: Estrutura básica para integração com o navegador.
